Personal notes

Margin: the extra filler "outside" the container. From the <- [element completed] ->
margin: A (ALl)
margin: A,B,C,D (TOP,RIGHT,BOT,LEFT)
margin: Y,X (TOP/BOT and LEFT/RIGHT)
possible set value: AUTO (horizontally center the element in it's container)



Padding: the extra filler "inside" the container. From the [<- element out ->]
same logic as above exclude AUTO


FlexBox:
Display: Flex
Allows container to be manipulated in Main-Axis and Cross-Axis. Directions are not fixed
to these terms. TOP/BOT/LEFT/RIGHT. Main-axis choose a direction (row/row reverse or col
col reverse) and Cross-Axis will be perpendicular to it.


Applies to All elements
justify-content: Modifies the Main-Axis
Align-item: Modifies the Cross-Axis

Flex: # # #;
Each Value is shorthand for flex-grow, flex-shrink, flex-basis
First:  if Positive, value growth
Second: if Positive, value shrink
Third: Starting base value of item
